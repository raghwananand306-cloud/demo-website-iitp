Problem Statement

Traditional antivirus systems depend on signature-based detection which fails to identify zero-day, polymorphic, and AI-generated malware. These attacks evolve rapidly and bypass static security systems easily.

Proposed Solution

This project uses Machine Learning models such as Random Forest, SVM, and Logistic Regression to classify files as malware or benign based on extracted features and behavioral patterns.
